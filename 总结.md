## CSS 相关

1. box-shadow 效果居然可以叠加的

   ```css
   * {
     box-shadow: 0 30px 50px rgba(0, 0, 0, 0.3), 0 20px 20px rgba(5, 226, 255, 0.3);
   }
   ```

2. 绝对定位 的值拉伸

   ```css
   .disk_cover {
     position: absolute;

     top: 0px;
     left: 0px;
     right: 0px;
     bottom: 0px;
   }
   ```

3. 背景也是可以在 padding 中显示的,且 padding 可以 被子元素当成宽高

   ```css
   .father {
     position: relative;
     width: 160px;
     height: 20px;
     padding: 80px;
     background: red;
   }
   .sun {
     position: absolute;

     width: 100%;
     height: 100%;
     background: black;
   }
   ```
